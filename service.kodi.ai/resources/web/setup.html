Kodi-AI · Setup

Setup over local network. Only continue if you trust your WiFi.
Step 1 of 4

OpenRouter API key

Already configured

OpenRouter is a single gateway to GPT, Claude and Gemini. Typical cost: $1–5/month for normal Kodi-AI use.

  1. Open openrouter.ai/keys
  2. Sign in, add $5 credit under Credits.
  3. Tap Create Key, copy it.
Step 2 of 4

Telegram bot

Already configured

You'll create your own Telegram bot. Kodi-AI talks to you through it.

  1. Open @BotFather in Telegram.
  2. Send /newbot, pick a name and username (must end in bot).
  3. Copy the token BotFather returns (digits:letters).
